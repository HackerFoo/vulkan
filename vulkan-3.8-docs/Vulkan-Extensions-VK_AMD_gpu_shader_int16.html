Description

This extension adds support for using 16-bit integer variables in shaders.

Deprecation by VK_KHR_shader_float16_int8

Functionality in this extension was included in VK_KHR_shader_float16_int8 extension, when PhysicalDeviceFeatures::shaderInt16 and PhysicalDeviceShaderFloat16Int8FeaturesKHR::shaderFloat16 are enabled.
